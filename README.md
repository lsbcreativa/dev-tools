# DevTools Online

**85+ free developer tools that run entirely in your browser.** No servers, no signups, no data collection.

Live at [toolboxurl.com](https://toolboxurl.com)

## Tech Stack

- **Next.js 16** with App Router and static export (`output: "export"`)
- **React 19** + TypeScript 5.9
- **Tailwind CSS 4** for styling
- **Lucide React** for icons
- **pdf-lib** for PDF manipulation (merge, split, compress)
- **exceljs** for Excel read/write
- **html-to-image** for code screenshots
- **isomorphic-dompurify** for XSS sanitization

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | Build static site to `out/` |
| `npm run lint` | Run ESLint |
| `npm run deploy` | Deploy to production |
| `npm run rollback` | Rollback to previous deployment |

## Privacy

Everything runs 100% client-side. No data leaves your browser. No cookies, no telemetry, no third-party scripts.
